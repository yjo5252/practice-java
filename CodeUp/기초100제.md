### 1006
%기호가 앞에 붙는 문자를 형식 지정자라고 한다. 그 위치에서 지정한 형식으로 값을 출력해준다.
따라서 퍼센트 %를 출력하기 위해서는 %%로 출력해주어야 한다.
```java
public class prob1006{
  public static void main(String[] args){
   System.out.println("\"!@#$%^&*()\"");
  }
}
```
